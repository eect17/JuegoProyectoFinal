## 📝 Descripción del Proyecto
Este es nuestro proyecto final de curso. Es un juego sencillo desarrollado en **C#** donde aplicamos conceptos básicos como:
* Estructuras de control (If, Else, Switch).
* Ciclos de repetición.
* Lógica de programación para principiantes.

## 📂 Estructura del Repositorio
* **JuegoProyectoFinal/**: Contiene todo el código fuente del juego.
* **JuegoProyectoFinal.slnx**: Archivo para abrir el proyecto en Visual Studio.
* **Documentación/**: manuel de usuario etc.
## 🚀 Cómo jugar
1. Descarga los archivos del repositorio.
2. Abre el archivo de solución (.slnx) en **Visual Studio**.
3. Presiona el botón **Iniciar** o la tecla **F5** para probar nuestro juego.
